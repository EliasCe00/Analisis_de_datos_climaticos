Escenario A - Análisis de Datos Climáticos

---

## Objetivo del proyecto

El objetivo de este proyecto es desarrollar un script en Python que permita analizar datos climáticos históricos obtenidos desde un archivo CSV.

A partir de estos datos se busca generar indicadores básicos de temperatura y representar gráficamente la evolución de la temperatura a lo largo del tiempo.

---

## Dataset utilizado

Se utiliza un dataset de temperaturas globales históricas en formato CSV.

### Características del dataset

* Datos de temperatura promedio global.
* Registros mensuales y anuales.
* Formato CSV.
* Datos provenientes del análisis climático GISTEMP.

Los archivos utilizados se encuentran dentro de la carpeta `/datos`.
